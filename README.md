# geekaltulator
React native calculator using react navigation, redux and RN 0.64

## functions
* Login
* historial
* calculator

## Packages
* react '17.0.1'
* react-native '0.64'
* react-navigations '^5.x.x'
* redux '4.1.0'

## how to install
1. download repo
2. npm i --save
3. npm run android
4. in other terminal npm run start
